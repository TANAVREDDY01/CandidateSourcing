
# Open-to-Work Candidate Finder (Vercel)

## Deploy Steps
1. Upload this project to GitHub
2. Login to Vercel
3. Import GitHub repo
4. Framework preset: Other
5. Deploy
6. Open deployed URL
7. Paste JD + location
8. Download CSV with up to 500 matched open-to-work profiles

## Notes
- Uses public-web style matching logic
- Filters by job title, experience, and location
- Ready for future scraping integrations
